# python 5번째 과제
### 강의 노트 6번 32~38
-------------------------------------------------------------------------------------------------

## 32번
1. 내가 좋아하는 숫자 5개씩 2줄을 적기.
2. c안에 합집합을 뜻하는 *|* 적기.
3. c2에 교집합을 뜻하는 *&* 적기.
4. c와 c2를 각각 print하기.

- 결과 : {'6', '4', '7', '5', '3', '2', '1'},
         {'4', '5', '3'}
------------------------------------------------------------------------------------------------------------------------------

## 33번
1. 내가 좋아하는 숫자 5개씩 2줄을 적기.
2. c안에 차집합을 뜻하는 *-*를 적기.
3. c2에 대칭 차집합을 뜻하는 *^*를 적기.
4. c와 c2를 각각 print하기.

- 결과 : {'1', '2'},
         {'6', '1', '7', '2'}
------------------------------------------------------------------------------------------------------------------------------

## 34번
1. 내가 좋아하는 숫자 5개 적기.
2. update 메서드와 같은 *|*를 적기.
3. 그 뒤에 100 적기.
4. a를 print하기.

- 결과 : {'1', '3', 100, '4', '5', '2'}
------------------------------------------------------------------------------------------------------------------------------

## 35번
1. a를 강의자료와 똑같이 선언하기.
2. intersection_update 메서드를 뜻하는 *&=*를 다음 세트와 같이 적고 print하기.
3. difference_update 메서드를 뜻하는 *-=*를 다음 세트와 같이 적고 print하기.
4. symmetric_difference_update 메서드를 뜻하는 *^=*를 다음 세트와 같이 적고 print하기.

- 결과 : {400, 500},
         {100, 200, 300},
         {800, 700, 100, 200, 600, 300}
------------------------------------------------------------------------------------------------------------------------------

## 36번
1. 강의자료와 똑같이 세트들을 적기.
2. a가 상위 집합 이면으로 *if*조건문 사용하기.
3. 근데 이 세트는 상위집합과 부분집합을 동시에 만족하기 때문에 동시라는 조건을 먼저 적어줘야 함.(?)
4. 동시라는 조건에 ==를 사용하기.
5. *elif*로 상위를 뜻하는 *issuperset*과 부분을 뜻하는 *issubset*를, 마지막은 *없음*이라고 적기.

- 결과 : 동시
------------------------------------------------------------------------------------------------------------------------------

## 37번
1. 내가 좋아하는 숫자 5개 적기.
2. 1000의 숫자를 추가해야 하므로 *add* 사용하기.
3. 마지막 값을 삭제해야 하므로 *pop* 사용한 후, print하기.

- 결과 : {'3', '4', 1000, '5', '2'}
------------------------------------------------------------------------------------------------------------------------------

## 38번
1. 1부터 100까지 숫자 중이므로 *1부터 101까지*의 범위임.
2. 3과 5의 공배수는 3과 5로 값을 각각 나누었을 때 *0*이 되어야 함.
3. 따라서 *x % 3 == 0*과 *x % 5 == 0*이 들어가야 함. 

- 결과 : {75, 45, 15, 90, 60, 30}
